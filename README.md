## FFFormFramework

### 场景

1. 你的boss让你做一个大量信息录入的界面，类似这样

   | 姓名         | 请输入您的姓名                  |
   | ------------ | ------------------------------- |
   | 性别         | 请输入您的性别                  |
   | 星座         | 请输入您的星座                  |
   | 出生日期     | 请选择您的出生日期              |
   | 出生地       | 请选择您的出生地                |
   | 是否有对象   | 请问您是否有对象 （checkbox）   |
   | 是否有房有车 | 请问您是否有房有车 （checkbox） |

2. 当然，所有的表单提交，表单展示界面。 我都可以满足你

3. 如果你的项目是信息管理系统，那么你就更离不开我了
